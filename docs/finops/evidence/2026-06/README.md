# June 2026 Cost Evidence

This directory contains a sanitized service-level derivative of Google Cloud Billing Reports for the platform's operating-cost monitoring.

The evidence coverage is 1-24 June 2026. This is a preliminary period-to-date snapshot, not a complete monthly invoice.

The source Billing Reports screenshot and CSV are external source artifacts and are not committed in this directory. Only appropriately redacted source evidence should be retained in the project's approved audit trail.

No billing account identifier, project identifier, account name, SKU-level information, resource name, Secret, credential, IP address, or connection string is committed here.
